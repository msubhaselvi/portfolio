Dynamic Ticket Fare Optimizer

PG Final Year Project | Team of 1

Dynamic Ticket Fare Optimizer is a web-based system that automatically predicts and adjusts ticket prices based on travel demand conditions. The system uses a Random Forest Regression machine learning model to calculate optimized ticket fares by analyzing factors such as simulated demand level, days remaining before travel, weekends, and festival seasons. It is developed using the Flask framework and includes features like user registration, secure login, ticket booking, admin dashboard, and automatic ticket generation. The system helps transportation providers optimize revenue while ensuring fair pricing for passengers.

Student Vault Management System

Internship Project: May 2025 – June 2025

Forecasting Flight Delays

PG 2nd Sem Project | Team of 2

Flight Delay Prediction System uses Random Forest Regression to estimate delays based on flight data like route, time, airline, and history. It features a Tkinter GUI for real-time predictions. The project helps improve airline planning and reduce passenger issues. Future enhancements include using LSTM deep learning for better accuracy.
